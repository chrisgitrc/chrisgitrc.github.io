Zydis Disassembler

I made a disassembler in C by wrapping the Zydis Engine.
I made a cli app that would take in a input file and use the linux inotify syscall to track that input files state. Whenever the file would be updated the Engine was called on the file after parsing. The output would then be printed to stdin. I made this to learn about opcodes for the x86-64 architecture. I learned how to make basic elf binaries and learning opcodes would make coding in and debugging the code segment faster and easier. This overall helped me learn about elf binaries, binary enumeration, and binary exploitation. This was also what led to my inspiratin of another project which was in making a debugger.
